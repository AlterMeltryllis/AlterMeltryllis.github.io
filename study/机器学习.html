# 创建新环境

现在我们安装好了Anconda，可以在其中创建各种不同的环境，每个环境可以在创建的时候使用不同的版本，这样当我们需要使用不同的版本运行机器学习的代码时可以方便切换。

下为例创建一个新的环境

在终端输入以下代码：

conda  create  --name  python38   python=3.8
1

其中name字段后面的内容是自己定义的能够区分就行，python3.8为指定的python版本，要是需要不同的python版本更改版本号就可以了。

在创建完成后在终端输入conda activate python38就可以启动该环境了，此时终端前会显示为：

(python38) l15hee7@L15hee7deMacBook-Pro ~ %
1

# 配置到pycharm

然后我们在跑代码的时候需要在pycharm中使用此环境，首先打开pycharm，在系统左上角选择settings

然后选择当前的project（为你的项目文件夹），然后选择python interpreter并点击右侧的添加local interpreter

选择左侧的Conda Environment，在Use existing environment选择你刚创建的部分即可。

此时你使用的为自己刚创建的环境。
