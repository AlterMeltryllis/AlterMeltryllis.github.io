# LaTex学习和使用

# 0x00 序

因为科研需求需要学习使用LaTeX所以开此文章记录安装的过程和学习使用技巧以及遇到的相关问题。

# 0x01 安装和配置

本人目前的系统为macOS 13.4.1 (c) intel芯片，安装可以选择：1.直接从官网 (opens new window)下载 2.使用homebrew下载，就方便来说使用homebrew应该是最快的，但是为了方便没折腾明白homebrew的人使用此处采用官网下载的方式

# 官网下载

打开官网 (opens new window)，点击MacTex即可开始下载，文件比较大下载的速度因人而异。

下载完之后一直点继续就可以了：

# homebrew下载

打开终端，输入：

brew install mactex --cask
1

即可下载和安装

然后安装一个LaTeX编辑器就行了，我使用的是TexStudio

# 0x02 TexStudio使用
