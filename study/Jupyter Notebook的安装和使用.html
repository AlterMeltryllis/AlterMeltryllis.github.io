# Jupyter Notebook的安装和使用

# 0x00

刚进研究生第一堂机器学习的课老师就说需要使用Jupyter Notebook所以先自己安装一下，和之前一样本机系统为macOS

# 0x01 安装

在终端输入代码pip install jupyter进行下载和安装，完成后在终端输入IPython -V，可以查看到当前安装的版本：

# 0x02 如何切换conda环境

因为我在安装的过程中想过个问题，就是我们使用conda创建了很多的镜像，那么在使用的时候我们如何切换不同conda环境呢？与此同时需要在每个conda环境中都安装jupyter吗？

带着这个疑问搜索了一下发现答案是不需要，只需在base环境中安装jupyter，其余的环境中只需要ipykernel就可以了。

首先需要本机base中安装好了jupyter，然后切换到你需要的环境（此处以python38为例）在终端输入conda activate python38激活环境。

conda activate python38
1

然后在环境中安装ipykernel

conda install  ipykernel
1

然后我就看到了两种不同的解决方案：

方案1 (opens new window)需要在每个conda中都输入conda install ipykernel然后将环境写入到notebook中：示例如下

python -m ipykernel install --user --name yourname --display-name "yourname"
1

然后在base中再执行jupyter notebook就行了

方案2 (opens new window)是这篇博客中的方案3

此方案他说是一键添加所有环境

conda activate my-conda-env    # this is the environment for your project and code
conda install ipykernel
conda deactivate

conda activate base      # could be also some other environment
conda install nb_conda_kernels
jupyter notebook
1
2
3
4
5
6
7

注意：这里的 conda install nb_conda_kernels 是在 base 环境下操作的。

安装好后，打开 jupyter notebook 就会显示所有的 conda 环境啦，点击随意切换。
